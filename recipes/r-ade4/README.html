r-ade4¶

Multivariate data analysis and graphical display.

Home http://pbil.univ-lyon1.fr/ADE-4, Mailing list: http://listes.univ-lyon1.fr/wws/info/adelist
Versions 1.7_2
License GPL (>= 2)
Recipe https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-ade4

Installation¶

With an activated Bioconda channel (see Setup and usage), install with:

conda install r-ade4

and update with:

conda update r-ade4
