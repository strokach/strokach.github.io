kraken-all
==========

This metapackage installs kraken with optional dependency jellyfish for building kraken databases.

======== ===========
Home     http://ccb.jhu.edu/software/kraken/
Versions 0.10.5beta
License  GPLv3
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/kraken-all
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`setup`), install with::

   conda install kraken-all

and update with::

   conda update kraken-all
