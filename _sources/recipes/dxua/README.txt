dxua

command-line tool for uploading files to the DNAnexus Platform

Versions 1.5.11
License  Apache v2.0
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/dxua

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`setup`), install with::

   conda install dxua

and update with::

   conda update dxua
