beagle-lib
==========

general purpose library for evaluating the likelihood of sequence evolution on trees

Versions 2.1.2

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`setup`), install with::

   conda install beagle-lib

and update with::

   conda update beagle-lib
