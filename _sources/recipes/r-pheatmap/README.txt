r-pheatmap
==========

Implementation of heatmaps that offers more control over dimensions and appearance.

======== ===========
Home     
Versions 1.0.8
License  GPL-2
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-pheatmap
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`setup`), install with::

   conda install r-pheatmap

and update with::

   conda update r-pheatmap
