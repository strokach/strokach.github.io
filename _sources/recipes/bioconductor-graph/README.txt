bioconductor-graph
==================

A package that implements some simple graph handling capabilities.

======== ===========
Home     http://bioconductor.org/packages/release/bioc/html/graph.html
Versions 1.48.0
License  Artistic-2.0
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-graph
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`setup`), install with::

   conda install bioconductor-graph

and update with::

   conda update bioconductor-graph
