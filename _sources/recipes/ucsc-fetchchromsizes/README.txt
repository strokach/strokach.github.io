ucsc-fetchchromsizes
====================

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`setup`), install with::

   conda install ucsc-fetchchromsizes

and update with::

   conda update ucsc-fetchchromsizes
