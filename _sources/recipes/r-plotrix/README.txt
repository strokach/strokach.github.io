r-plotrix
=========

Lots of plots, various labeling, axis and color scaling functions.

Home

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`setup`), install with::

   conda install r-plotrix

and update with::

   conda update r-plotrix
