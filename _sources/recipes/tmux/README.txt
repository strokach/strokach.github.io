tmux

tmux is a terminal multiplexer.

Home     http://tmux.sourceforge.net

Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/tmux

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`setup`), install with::

   conda install tmux

and update with::

   conda update tmux
